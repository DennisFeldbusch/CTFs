# General

* running `file chall` you get the follwing info: 
    1. ELF 64-bit executable
    2. PIE<sup>[1](#pie)</sup>
    3. stripped -> debug information missing 
* to debug the file this [docker image](https://github.com/DennisFeldbusch/RevEng-Docker) where used
